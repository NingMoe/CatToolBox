# CatToolBox
## 微信小程序简介

<img src="img/logo.jpg">

### 当前版本：v0.9.6

猫咪工具箱是一款集结万千花样于一体的小程序，目前功能还在不断完善中，会不定期更新功能~

## 功能列表

- 物流查询
- 星座查询
- 聊天机器人
- 笑话大全
- 黄历运势
- 音乐播放
- 历史查询
- 每日壁纸
- 颜文字
- 计算器
- 文字翻译
